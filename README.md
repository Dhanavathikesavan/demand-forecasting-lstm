Demand Forecasting using LSTM

Project Description:
This project predicts future demand using an LSTM model.

Files Included:
- demand_forecasting_lstm.py
- supermarket_demand_data.csv
- requirements.txt

Steps to Run:
1. Install libraries:
   pip install -r requirements.txt

2. Run:
   python demand_forecasting_lstm.py

Output:
- Graphs and performance metrics

Note:
Synthetic dataset is used.
